Design, build, and use geospatial applications

A workflow that provides geospatial applications to support monitoring and managing heterogeneous agricultural landscapes has been designed and developed. Where possible, this workflow builds on existing software (e.g. QField mobile GIS). If existing software is not available then new applications were developed (e.g. Express.js server-side apps for syncing data collected in the field).
