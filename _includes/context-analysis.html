Context Analysis

Generate an understanding of the environment in which an ICT application will be deployed and used

Participatory modelling, household surveys, and expert interviews were collected to generate a detailed understanding of how the landscape is used to support livelihoods. This informs development of geospatial tools tailored for collecting and visualising geospatial data at scales that reflect how (all) people use the landscape.

The technical features of over 100 mobile data collection applications were evaluated to identify functionalities provided by existing software that could be used in geospatial tool. The GIS tools used by the Fijian government, Pacific, and international organisations were reviewed to avoid duplication.

Fine spatial resolution landscape dynamics maps do not exist for Pacific Island Countries. A machine learning approach to map inter-annual land cover dynamics using Sentinel-2 remote sensing data has been developed.
